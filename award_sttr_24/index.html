STTR project awarded from Navy

Real-Time In-Flight Aircraft State Estimation

Published: Sep 30, 2024 by Daning Huang

Together with STI, we are awarded a Navy STTR Phase I project, “Real-Time In-Flight Aircraft State Estimation”, that aims to enhance the capability to estimate the states and parameters of an aerial system in-flight using a data-augmented state estimation algorithm. This is a spin-off from our physics-infused reduced-order modeling methodology from the NSF CAREER project.
